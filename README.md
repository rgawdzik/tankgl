# tankgl

> A webgl game.

### Getting started

`npm install tankgl`

### Testing

watches for file changes and reruns tests each time
```bash
  $ grunt watch
```

runs spec tests
```bash
  $ grunt test
```

produces coverage report (needs explicit piping)
```bash
  $ grunt cov
```

###Building
To build simply:
```bash
  $ grunt build
```

###Linting
To lint simply:
```bash
  $ grunt lint
```
